1. Giới hạn độ dài tên sản phẩm  
    max-width: 220px;
2. Để text không xuống dòng  
    white-space: nowrap;
3. Xử lý text bị tràn ra khỏi box 
    overflow: visible - mặc định nd tràn ra ngoài
    overflow: hidden - phần tràn bị cắt
    overflow: scroll - luôn có thanh cuộn
    overflow: autpo - có thanh cuộn khi cần 
4. Quy định cách hiển thị khi text bị cắt 
    clip - cắt thẳng
    ellipsis - ... 
    vd: text-overflow: ellipsis;
